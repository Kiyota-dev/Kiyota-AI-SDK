# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, and contains a way to track changes for this repository. Read the [documentation](https://github.com/changesets/changesets/blob/main/README.md) for more details.

We use Changesets to version and publish the `@nurovia/*` packages. Add a changeset for any change that affects a publishable package by running `pnpm changeset`.
